# Submission Plan

## Recommended Positioning

AI Idea Radar is a planning assistant for AI builders. It helps users turn vague AI ideas into public project plans with a clear MVP, README structure, risk notes, and launch checklist.

## Why It Fits An AI Project Activity

- The product is about AI project creation and planning.
- It can later integrate an OpenAI-compatible API for richer generation.
- The first version is reviewable without accounts, backend setup, or paid services.
- The README and demo are simple enough for judges and GitHub visitors to understand quickly.

## Next Steps Before Publishing

1. Create a new public GitHub repository.
2. Push this project as the initial version.
3. Enable GitHub Pages for `index.html`.
4. Add the official DreamSeed README banner/template if required by the platform.
5. Submit the public repository URL for review.

## Star Strategy

- Share a screenshot of the generated project radar output.
- Position it as a tool for makers joining AI hackathons or public project contests.
- Ask builder communities for feedback first, then ask for Stars after the demo is useful.
