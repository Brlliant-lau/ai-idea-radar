# AI Idea Radar

AI Idea Radar is a lightweight browser tool that turns a rough AI product idea into a practical project brief, MVP scope, risk map, README outline, and launch checklist.

It is designed for AI makers who want to publish small but complete GitHub projects quickly, especially for public AI project activities where reviewers need to understand the project in a few minutes.

## Why This Exists

Many AI projects fail before shipping because the idea is too broad, the target user is vague, or the README does not explain the value clearly. AI Idea Radar provides a guided planning cockpit so a maker can move from "interesting idea" to "buildable public repo" faster.

## Features

- Converts a raw idea into a one-line project positioning statement.
- Suggests an MVP scope based on the selected AI capability.
- Highlights practical risks such as hallucination, review needs, privacy, and automation safety.
- Produces a README outline suitable for a public GitHub project.
- Includes a DreamSeed-style launch checklist for public AI project submission.

## Demo

Open `index.html` in a browser. No build step, backend, or API key is required for the first version.
